# Agenix v0.1.0 Release Notes

## Status

Agenix v0.1.0 is the first complete reference-runtime release. It proves the
local closed loop for reusable agent skills:

- load a declarative skill manifest
- execute through runtime-controlled tools
- enforce local policy
- record a trace
- run verifiers
- package the skill as a portable capsule
- publish and pull through the local registry
- replay and re-verify the recorded run

## Release Gate

Run from the repository root:

```bash
go run ./cmd/agenix acceptance
go test -count=1 ./...
go vet ./...
go build ./cmd/agenix
```

The canonical V0 gate is also mirrored by GitHub Actions in
`.github/workflows/v0-release-gate.yml`.

## Included Surface

- `agenix build`

- `agenix validate`
- `agenix publish`
- `agenix pull`
- `agenix registry list/show/resolve`
- `agenix acceptance`

## Canonical Skills

- `repo.fix_test_failure`
- `repo.analyze_test_failures`
- `repo.apply_small_refactor`

## Intentional Exclusions

v0.1.0 is a local reference runtime. It does not claim:

- strong OS sandboxing
- a remote executor
- a hosted or trusted registry
- artifact signatures
- OCI distribution
- default provider-backed adapter coverage

The provider-backed `openai-analyze` path remains an opt-in smoke path outside
the V0 release gate.
