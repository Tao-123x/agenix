# Agenix v0.1.0 Release Notes

## 状态

Agenix v0.1.0 是第一个完整的 reference-runtime release。它证明了本地 agent
skill 复用闭环：

- 加载声明式 skill manifest
- 通过 runtime 控制的工具执行
- 执行本地 policy
- 记录 trace
- 运行 verifier
- 把 skill 打包成可移植 capsule
- 通过本地 registry publish / pull
- replay 并重新 verify 已记录的 run

## Release Gate

在仓库根目录运行：

```bash
go run ./cmd/agenix acceptance
go test -count=1 ./...
go vet ./...
go build ./cmd/agenix
```

canonical V0 gate 也会由 GitHub Actions 的
`.github/workflows/v0-release-gate.yml` 执行。

## Canonical Skills

- `repo.fix_test_failure`
- `repo.analyze_test_failures`
- `repo.apply_small_refactor`

## 有意排除

v0.1.0 是本地 reference runtime。它不声称提供：

- 强 OS sandbox
- 远程执行器
- 托管或可信 registry
- artifact 签名
- OCI 分发
- 默认 provider-backed adapter 覆盖

provider-backed `openai-analyze` 路径仍然是 opt-in smoke path，不属于 V0
release gate。
